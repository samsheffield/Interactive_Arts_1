HOMEWORK

HOMEWORK 3: NARRATIVE DEVICE

Create a Processing sketch that responds to input (keyboard or mouse) and uses images and text to describe an event in your life.

  • The narrative can be linear or non-linear.
  • The content can be specific or atmospheric.
  • Any combination of input (keyboard/mouse) and output (images/text) is acceptible.
